### How to add the RNP logo:

1. **Save the Rwanda National Police logo image** (the blue circular badge with white dove and olive branches)

2. **Name it exactly:** `rnp-logo.png`

3. **Place it in this folder** alongside this README file

4. **The full path should be:**
   ```
   /home/jambo/New_Traffic_Project/government-dashboard/public/assets/rnp-logo.png
   ```

5. **After saving, refresh your browser** - the logo will automatically appear in:
   - Navigation bar (top left)
   - Hero section (center)
   - Background watermarks
   - Footer
   - Login page
   - Register page
   - Browser tab icon (favicon)

### Recommended Format:
- **Format:** PNG with transparent background
- **Size:** At least 512x512px for best quality
- **File name:** Exactly `rnp-logo.png` (case-sensitive)
